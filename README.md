# Portfolio v1

A minimal animated personal portfolio with a built-in music player and cinematic background.

## Features

- Fullscreen background video
- Frosted glass hero card
- Interactive music player with playlist
- Custom volume slider
- Animated social media icons
- Intro click-to-enter overlay

## Preview

Clean layout with centered hero section, profile avatar, social links and embedded music controls.

## Music System

The music player supports multiple tracks.  
To change songs:

1. Add your `.mp3` files inside the `mp3` folder
2. Edit the `playlist` array inside `script.js`
3. Update the track title

## Customization

You can easily customize:

- Profile image (`img/`)
- Background video (inside `index.html`)
- Social links
- Colors and design inside `style.css`

No frameworks used — pure HTML, CSS and JavaScript.
